Для того чтобы проект заработал нужно импортировать библиотеки:
1) pyJWT
2) django-cors-headers

Добавлены некоторые настройки в settings.py для корректной работы

Для запуска проекта придется удалить БД и миграции а затем повторить процедуру миграции: makemigrations, migrate.
Обратите внимание: сначала вводите командy: makemigrations а потом только migrate!.
